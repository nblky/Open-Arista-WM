&emsp;&emsp;开源虚拟无线控制器<br>
&emsp;&emsp;Open Virtual Wireless Manager<br>

## 名字来源:
&emsp;&emsp;H3C 无线控制器叫做AC，他有基于虚拟化部署的无线控制器叫做VAC，所以项目取名OpenVWM

## 项目起源:
&emsp;&emsp;因本人捡“电子垃圾”上瘾，在2024年8月某天在TB下单了一款已经停服的监控产品，这一放就是一年多。由于裸辞在家无聊，继而研究起了那个摄像头。靠着商品链接中买家的评论和AI加持下，成功伪复活了该款设备的服务。那个项目结束后，想起了家里角里已经躺了半年多的Arista的AP（因为Arista AP没有类似Aruba 的胖瘦一体模式，只能靠自家CVP/CV-CUE进行管控），我就想着靠AI尝试一下，能否自建无线控制平台。
